批量生成缩略图

    
X

在淘宝店面的产品图片设计中，经常会制作大量缩略图。本节将演示如何使用脚本，批量缩放和导出缩略图。

首先创建一个空白的脚本文档，并保存在硬盘上某个位置。

接着输入脚本代码：

//定义一个变量[size]，作为缩略图的尺寸。
//您可以优化此处的逻辑为:弹出一个输入窗口，提示用户输入一个数字做为缩略图的尺寸。
var size = 150;

//调用[Folder]的[selectDialog]命令，弹出文件夹选择窗口，提示用户选择待处理的文件所在文件夹。
//并将文件夹存储在变量[inputFolder]中。
var inputFolder = Folder.selectDialog("请选择导入的文件夹：");

//调用[Folder]的[selectDialog]命令，弹出文件夹选择窗口，提示用户选择缩略图输出的文件夹。
//并将文件夹存储在变量[outputFolder]中。
var outputFolder = Folder.selectDialog("请选择输出的文件夹：");

//定义一个变量[fileList]，表示导入文件夹下的所有图片。
var fileList = inputFolder.getFiles();

//添加一个for循环，遍历所有图片。
for (var i = 0; i < fileList.length; i++)
{

     //判断如果图片是正常文件，并且是非隐藏模式，则继续后面的操作。
     if (fileList[i] instanceof File && fileList[i].hidden == false) 
    {
		
        //打开遍历到的图片，并赋予变量[doc]。
        var doc = app.open(fileList[i]);

        //如果遍历到的图片的宽度大于其高度，则继续下面的动作。
        if(parseInt(doc.width) > parseInt(doc.height))
        {
            //由于缩略图的宽大于高，所以设置缩略图的宽度为变量[size]，高度则按比例缩小。
            var newHeight = parseInt(doc.height) * size / parseInt(doc.width);

            //将图片按照计算得出的宽度和高度进行缩放。
            doc.resizeImage(size, newHeight);
        }
        else
        {
            //接下来，处理图片高度大于宽度的情况。处理逻辑与宽度大于高度的情况是一致的。
            //由于缩略图的高大于宽，所以设置缩略图的高度为变量[size]，宽度则按比例缩小。
            var newWidth = parseInt(doc.width) * size / parseInt(doc.height);

            //将图片按照计算得出的宽度和高度进行缩放。
            doc.resizeImage(newWidth, size);
        }
		
        //此时图片的尺寸有可能不是正方形。如果我们的缩略图需要保持正方形的话，还需要设置为画布尺寸。
        doc.resizeCanvas(size, size, AnchorPosition.MIDDLECENTER);

        //定义一个变量[file]，作为缩略图导出的路径。
        var file = new File(outputFolder + "/output" + i + ".jpg");

        //定义一个变量[jpegoptions]，作为缩略图输出的图片格式。
        var jpegOptions = new JPEGSaveOptions();

        //调用[doc]对象的[saveAs]命令，保存当前缩略图到指定路径上。
        doc.saveAs(file, jpegOptions, true, Extension.LOWERCASE);

        //操作完成后，关闭当前缩略图文档。
        doc.close(SaveOptions.DONOTSAVECHANGES);
    }
}

我们的脚本已经编写完成，把它保存下来。

然后切换到Photoshop。

接着依次点击[文件] > [脚本] > [浏览]，打开刚刚保存的脚本文件。

在弹出的[载入]窗口中，直接双击脚本名称，Photoshop将直接调用并执行该脚本。

在弹出的[选择文件夹]窗口，选择待处理文件所在的文件夹。然后点击[确定]按钮，确认文件夹的选择。

在弹出的[选择文件夹]窗口，选择缩略图输出的文件夹。然后点击[确定]按钮，Photoshop开始批量生成缩略图。
